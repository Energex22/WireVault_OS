WireVault Alpha 0.9.6.1 Home System Card Hotfix

Required baseline: Alpha 0.9.6

1. Copy the js and css folders into your WireVault_OS repository.
2. Replace js/views/home-view.js and css/views.css.
3. Restart WireVault Core.
4. Press Ctrl+F5.

This hotfix replaces the old Home System Health widget with the full Raspberry Pi card and removes the separate CPU Temperature, Memory, and Storage widgets.
