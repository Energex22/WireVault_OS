WireVault Alpha 0.8.2 Folder Selector Hotfix

Required baseline: Alpha 0.8 or Alpha 0.8.1

1. Close the WireVault Core command window.
2. Copy the backend folder from this patch into your WireVault_OS repository.
3. Replace backend/wirevault_core.py.
4. Run start_core_windows.bat again.
5. Hard-refresh the browser with Ctrl+F5.
6. Open Files and press Browse.
