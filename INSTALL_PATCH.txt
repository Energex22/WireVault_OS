WireVault Alpha 1.5.2 — Collapsed Profile Editor and Full Accent Theme

Required baseline: Alpha 1.5.1

1. Close WireVault Core and every WireVault browser tab.
2. Copy all patch contents into the repository root.
3. Replace the existing files.
4. Restart WireVault Core.
5. Open WireVault in a fresh tab.

- Profile accent colors now theme buttons, fills, glows, active states, progress bars, highlights, and panels.
- The duplicate Accent Color setting is hidden because profiles now own it.
