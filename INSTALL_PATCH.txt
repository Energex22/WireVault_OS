WireVault Alpha 0.9.4 — Quiet Media and No-Flash Navigation

Required baseline: Alpha 0.9.3

1. Close WireVault Core.
2. Copy all folders from this patch into your WireVault_OS repository.
3. Replace the four existing files.
4. Start WireVault Core again.
5. Press Ctrl+F5 in the browser.

Behavior after this patch:
- Media scans once when WireVault starts.
- Additional scans only happen when you press Scan.
- Routine successful scans do not enter Notification Center.
- Scan errors can still notify you.
- Page switching is instant with no full-page flash.
- Card, dock, panel, toast, focus, and boot animations remain.
