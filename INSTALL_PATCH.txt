WireVault Alpha 1.4 — User Profiles Foundation

Required baseline: Alpha 1.3 Home Layout Editor

1. Close WireVault Core and all browser tabs.
2. Copy all patch contents into your repository root.
3. Replace the existing files.
4. Restart WireVault Core.
5. Open WireVault in a new browser tab.

Added:
- Profile button in the top status area.
- Profile switcher panel.
- Create and delete local profiles.
- Separate Home widget layouts per profile.
- Active profile is remembered.

Profiles are local-browser only in this foundation release. A later backend update can synchronize them between devices.
