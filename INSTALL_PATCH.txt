WireVault OS Alpha 0.6 — Notification Center Patch

Required baseline:
WireVault OS 4.0 — Alpha 0.5 Dynamic Backgrounds

Apply the patch:
1. Open your local WireVault_OS repository folder.
2. Copy everything inside this patch folder into the repository root.
3. Choose Replace when Windows asks about existing files.
4. Run start_core_windows.bat.
5. Open http://127.0.0.1:8080.
6. Test the notification button in the upper-right status area.
7. Commit and push the changes.

- js/services/core-api-service.js
